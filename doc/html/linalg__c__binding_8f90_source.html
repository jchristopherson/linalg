413  !! B = alpha * A**T * A + beta * B is to be calculated; else, set to false
414  !! if A is lower triangular, and B = alpha * A * A**T + beta * B is to
415  !! be computed.

417  !! @param[in] alpha A scalar multiplier.
418  !! @param[in] a The N-by-N triangular matrix. Notice, if @p upper is true
419  !! only the upper triangular portion of this matrix is referenced; else,
420  !! if @p upper is false, only the lower triangular portion of this matrix
421  !! is referenced.
